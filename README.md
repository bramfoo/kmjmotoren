# KMJ Motoren

Website for KMJ Motoren, a scooter and motor service business based in Den Haag.

## Stack

Static single-page site — plain HTML, CSS, and vanilla JavaScript. No build tools or dependencies.

## Structure

```
index.html   — page content and layout
style.css    — all styles, including responsive breakpoints
main.js      — mobile nav toggle and contact form handling
```

## Running locally

Open `index.html` directly in a browser, or serve it with any static file server:

```bash
npx serve .
```
